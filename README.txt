Это бот для учета ремонтов, перепродаж и рабочих диалогов с Avito.

Здесь уже можно:
-заводить ремонты и 💸 перепродажи
-вести активные, закрытые и архивные заказы
-добавлять расходы, предоплаты и финальные суммы
-редактировать заказ: модель, комментарий, клиента и цены
-хранить клиента с ФИО, телефоном, Telegram и историей обращений
-искать по ID, модели, ФИО, номеру и Telegram клиента
-ставить этапы, 🚨 приоритет и ⏰ напоминания
-смотреть статистику по устройству и по месяцам
-получать Avito-чаты, отвечать из бота и создавать заказ из диалога
-использовать шаблоны быстрых ответов для Avito
-настраивать Avito, рассылку и при необходимости очищать свои данные

## Админка

Админу доступны:
- просмотр пользователей;
- бан и разбан;
- рассылка сообщений;
- изменение части текстов прямо из Telegram;
- остановка бота и обновление через git;
- просмотр общей активности по пользователям.

Запуск бота: 
-Создаем в папке файл .env и запускаем его через удобный редактор
-Меняем свой BOT_TOKEN и ADMIN_ID
-закидываем папку на свой vds-сервер 
-Устанавливаем, обновляем python, и запускаем на vds
         cd PerekupBOT
         sudo apt update
         sudo apt install python3-venv
         python3 -m venv venv
         source venv/bin/activate
         pip install --upgrade pip
         pip install -r requirements.txt
         python run.py
